Resources

Curated, verifiable sources on Mukundara Hills Tiger Reserve (MHTR): official portals, plans and notifications, scientific studies, maps, citizen-science datasets, and carefully chosen context pieces.

Preference: primary sources (government notifications, official plans, peer-reviewed research). Media articles are included for context, not as sole evidence.

How to use this page

  • Start with: official portals, ESZ / clearance documents, and management / master plans.
  • Then read: research papers for methods, sampling effort, and species/habitat detail.
  • Use maps: for general orientation only—avoid sharing sensitive coordinates.
  • Cross-check: if something is a big claim, it should survive multiple sources.
